# 📚 Prompt Library

A curated collection of Claude prompts, built with HTML and hosted on GitHub Pages.

🔗 **Live site:** [tahmeedmustafa.github.io/promt-library](https://tahmeedmustafa.github.io/promt-library/)

## About

Prompt Library is a lightweight, browsable collection of prompts for use with Claude. No frameworks, no build step — just HTML, CSS, and a passion for well-crafted prompts.

## Getting Started

Clone the repo and open `index.html` in your browser:

```bash
git clone https://github.com/tahmeedmustafa/promt-library.git
cd promt-library
open index.html
```

Or just visit the [live site](https://tahmeedmustafa.github.io/promt-library/).
